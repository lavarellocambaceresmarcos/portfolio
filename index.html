Diseñador web

Sobre mi

  • Aprendí a codear a fines de 2020 y desde entonces no paro. Soy tutor de Desarrollo Web, maquetador y diseñador UX/UI y actualmente estoy en proceso de convertirme en desarrollador de software.
  • Manejo HTML5, CSS3, Bootstrap, Sass y estoy estudiando Javascript para poder combinar mi entusiasmo por el diseño con el de la programación.
  • Como objetivo laboral, me gustaría incorporarme a una empresa para seguir creciendo y desarrollando mis habilidades, o bien colaborar de manera freelance en diferentes proyectos.

Herramientas

Proyectos

    Desarrollo Web - Portfolio

  • Mi primer sitio web: diseñé y desarrollé este portfolio para el curso de Desarrollo Web en Coderhouse, utilizando HTML5, CSS3, Bootstrap, Sass y algunos detalles funcionales con Javascript.

    La Previa - Landing Page

  • Basándome en la investigación y el diseño de La Previa, realicé una landing page en Wordpress. Usé una combinación de Elementor y CSS propio para darle los estilos.

    UI Challenge - Signup

  • Diseño para Daily UI Challenge: pensé un formulario de inicio de sesión para una app de fotografía, además del diseño del logo.

    BALANZ - Rediseño de UI

  • Propuesta de rediseño para el sitio web de Balanz a partir de su imagen e identidad de marca, respetando la paleta de colores y el estilo general.

    UI Challenge - Checkout

  • Diseño para Daily UI Challenge: pantalla de pago con tarjeta. En este challenge, intenté ser lo más auténtico posible, creando las tarjetas de pago por mi cuenta, y seleccionando una paleta de colores, algo distinta a lo convencional.

    La Previa - UX/UI

  • A lo largo de tres meses en el curso de Diseño UX/UI de Coderhouse, me dediqué a diseñar este proyecto: una app de delivery online de bebidas; esta idea de proyecto surgió debido al aislamiento preventido y obligatorio, provocado por la pandemia. El fin era desarrollar una app que le permita al usuario comprar, a precio mayorista, sus bebidas favoritas, sin tener que salir de su domicilio. Realicé un benchmarking para analizar la competencia, además de empatizar, idear, prototipar y testear con usuarios.

    UI Challenge - Music App

  • Diseño para Daily UI Challenge: aplicación de música. Para este challenge, utilicé un gradiente entre tonos violetas y celestes, logrando así un contraste armonioso con el color principal de la app.

    Diseño UI - Delivery web page

  • Diseño para Daily UI Challenge: diseño de una landing page, para una empresa ficticia de delivery de comida. Me basé en la temática clásica de un sitio de comida rápida, para adaptarla con un estilo mucho más moderno y navegable.

    Frontend Challenge - Order Summary

  • Tomé un desafío de Front End Challenge, y lo adapté según la temática que se me indicó. La idea, era crear una card, siguiendo la temática de "Order Summary".
